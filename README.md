# CatTails 🐾

Progetto di apprendimento realizzato con **Godot Engine** seguendo vari tutorial e sperimentando meccaniche base per giochi 2D.

Obbiettivi di apprendimento:
- Struttura di un progetto
- Gestione delle scene
- Tileset
- Macchine a stati
- Interazione tra oggetti
- Vertex shaders
- Y-sorting
- NPC
- UI
- Inventario
- Menù

## Tecnologie

- 🎮 Engine: Godot
- 🧠 Linguaggio: GDScript
